# AI Chatbot Project

This is a beginner-friendly Python chatbot that responds to user input using keyword matching, conditional logic, random responses, and basic cybersecurity tips.

## Features
- Greets the user
- Gives cybersecurity tips
- Gives password safety tips
- Responds to basic questions
- Uses Python functions, conditionals, loops, and random responses

## How to Run
Download the file and run:

python ai_chatbot.py
